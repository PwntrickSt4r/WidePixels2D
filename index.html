WidePixels2D - Almost a native port of SEGA Mega Drive with Widescreen.

Fast, Smooth and Simple. These 3 words is fully describing this project :D

ROM Shown on video is Sonic 1: Widescreen Edition.

To run a game you need to legally acquire any of SEGA Mega Drive ROM's and rename them to .bin format and put into the "content" folder.I recommend you to get these from Steam if you previously bought original Sonic 1, 2, 3&Knuckles. To get widescreen you need to patch ROM's with Sonic Wide Autopatcher.

Current Version is v1.1.0
