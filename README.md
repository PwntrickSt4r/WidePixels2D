# WidePixels2D
Lightweight C++/SDL2 native runtime for Genesis Plus GX Wide.
> [!NOTE]
> The source code version in this repository is version 1.0.0. The most current version of the project is version 1.1.0, which will be distributed as a closed source and will be available for free download only from our website linked to the repository. This decision was made due to the hard difficulty of presenting the source code in a clear and accessible manner. You can download the current source code from the repository and fork it if you wish.
